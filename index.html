Herzliche willcommen!

Вашему вниманию представлено три страницы.

На первой из них, Karten, вам предстоит проверить своё знание неправильных глаголов, указав правильную форму. Для этого перетаскивайте карточки с буквами в ячейки.

На второй, Sortierung, вам надо рассотрировать слова по двум спискам так, чтобы в каждом списке были слова по одной тематике.

Третья же страница, Notizen, предназначена для заметок. Оставляйте здесь пометки или записывайте слова, которые хотите потом повторить. Размер поля для заметок можно менять.
